# AI-Powered-Talent-Scouting-Engagement-Agent


An intelligent AI system that automates candidate discovery, matching, and engagement.
It parses job descriptions, finds relevant candidates, simulates recruiter outreach, and returns a ranked shortlist based on **Match Score** and **Interest Score**.

---

## Problem Statement

Recruiters spend hours:

* Manually screening resumes
* Matching skills with job descriptions
* Following up with candidates

This project automates the entire pipeline using AI.

---

## Features

* Job Description Parsing (AI-powered)
* Candidate Matching with Explainability
* Conversational Engagement Simulation
* Interest Scoring using LLM
* Final Candidate Ranking
* Streamlit UI for interaction
* FastAPI backend (production-ready)

---

## System Architecture

```
User (Streamlit UI)
        ↓
FastAPI Backend (/process)
        ↓
LangGraph Pipeline
 ├── JD Parser (LLM)
 ├── Matcher (Rule-based)
 ├── Engagement (LLM)
 ├── Interest Scorer (LLM)
 └── Ranker (Final Score)
        ↓
Ranked Candidate Output
```

---

## Tech Stack

* **Backend:** FastAPI
* **Orchestration:** LangGraph
* **LLM:** Groq (LLaMA3)

## Scoring Logic

### Match Score

Based on:

* Skill overlap (60%)
* Experience match (40%)

### Interest Score

Generated by LLM based on simulated conversation

### Final Score

```
Final Score = 0.7 * Match Score + 0.3 * Interest Score
```

## Setup Instructions (Local)

### 1️ Clone Repo

```bash
git clone <your-repo-url>
cd AI-Powered-Talent-Scouting-Engagement-Agent
```

---

### 2️ Create Virtual Environment

```bash
python -m venv myenv
myenv\Scripts\activate   # Windows
```

---

### 3️ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4️ Add Environment Variables

Create `.env` file:

```env
GROQ_API_KEY=your_api_key_here
```

---

### 5️ Run Backend

```bash
uvicorn backend2.main:app --reload
```

Open:

```
http://127.0.0.1:8000/docs
```

---

### 6️ Run Frontend (Streamlit)

```bash
streamlit run app.py
```

Open:

```
http://localhost:8501
```

##  API Usage

### Endpoint:

```
POST /process
```

### Request:

```json
{
  "jd_text": "Looking for a Python ML engineer with FastAPI and 3 years experience"
}
```

### Response:

```json
{
  "top_candidates": [
    {
      "candidate": {
        "name": "Rahul Sharma",
        "skills": ["Python", "FastAPI"],
        "experience": 3
      },
      "match_score": 0.85,
      "interest_score": 0.75,
      "final_score": 0.81
    }
  ]
}
```
